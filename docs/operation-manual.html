个人知识产品网站操作手册

面向个人品牌、项目管理课程、AI工具资源库、会员内容与长期知识资产运营的建站说明。

1. 项目定位

这个网站不要定位成“个人主页”，而应定位成：

个人知识产品平台

更具体一点，是：

个人品牌官网
+ 项目管理课程平台
+ AI工具资源库
+ 课程资料下载中心
+ 会员知识服务系统

你的核心资产不是网站本身，而是 D:\Sy课程开发 里面长期积累的课程、PPT、文章、模板、案例、AI工具周报、教学资源和培训资料。

网站只是这些内容资产的展示入口、检索入口和商业化入口。

2. 总体建设原则

原则一：内容资产优先，页面设计其次

不要先想“我要几个栏目”，而要先想：

我有哪些资源？
每个资源属于什么主题？
适合谁看？
免费用户看到什么？
会员用户获得什么额外价值？
以后能不能搜索、复用、自动生成页面？

原则二：免费告诉用户是什么，会员告诉用户怎么做

免费：知道有什么
会员：知道怎么用、怎么做、怎么落地

例如 AI 工具，免费用户看到工具名称、官网、一句话介绍和适用方向；会员用户看到最佳入口、国内可用情况、注册方法、使用技巧、Prompt、教学案例、企业应用建议、替代工具、注意事项和视频教程。

这样用户不会觉得你在“故意藏东西”，而会觉得会员买的是经验、效率和整理能力。

7. 内容格式建议

长期来看，不建议直接维护大量 HTML 文件。例如 teacher-ai-weekly-report.html 可以先作为素材保留，但后续建议转成 Markdown、JSON，或“YAML元数据 + Markdown正文”。

2026-07-02-ai-weekly.md

内容结构可以是：标题、日期、分类、标签、摘要、正文、工具卡片、会员内容、相关资源。这样以后网站可以自动生成页面，维护成本低很多。

12. 搜索系统是核心功能

对于你这种资料量可能很大的项目，搜索比栏目更重要。理想体验是：

搜索：风险登记册

结果出现：
相关文章
PPT课件
Excel模板
课程章节
企业案例
AI工具
会员下载

第一阶段可以先做简单搜索。第二阶段再做高级搜索：按类型筛选、按主题筛选、按权限筛选、按更新时间筛选、按推荐等级筛选。

13. 会员系统建议

第一阶段不急着做完整会员系统。建议分三步：

第一步：公开静态网站、免费资源、课程介绍、资源目录
第二步：注册登录、会员页面、资源下载、搜索、收藏
第三步：支付、权限管理、学习记录、AI助教、知识库问答

技术上可以考虑 Cloudflare Pages、Supabase Auth、Supabase Database、Cloudflare R2 / Supabase Storage。

公开内容可以放在 GitHub。会员资源不要直接放 GitHub 公开仓库。会员资源建议放 Cloudflare R2、Supabase Storage 或其他对象存储。

14. 技术路线建议

第一阶段：
Astro
Markdown
GitHub仓库
GitHub Pages 或 Cloudflare Pages

原因：适合内容网站、支持 Markdown、SEO 好、速度快、后续可扩展。

部署优先 Cloudflare Pages；备选 GitHub Pages、Vercel、Netlify。如果只是公开静态网站，GitHub Pages 足够。如果以后要会员、权限、表单、边缘函数，Cloudflare Pages 更适合长期发展。

15. 三阶段建设路线

第一阶段：内容整理与静态网站

周期：1到2周。

完成资料盘点
建立内容资产登记表
搭建首页
搭建课程中心
搭建AI资源中心
搭建下载中心
上线公开版网站

产出：内容分类体系、网站栏目结构、第一版静态网站、10到30个精选公开资源页面。

第二阶段：搜索与会员雏形

周期：2到4周。

增加站内搜索
增加标签系统
增加资源卡片
增加会员可见内容
增加下载中心
增加注册登录

产出：AI工具导航、AI工具周报系统、课程资料库、会员内容页面。

第三阶段：AI知识库与自动化

后续持续建设，目标包括接入AI问答、基于课程资料构建知识库、自动推荐课程、自动生成资源专题、AI助教、学习路径推荐。

17. 当前最推荐的执行顺序

1. 先不急着做会员系统
2. 先整理 D:\Sy课程开发 的内容资产
3. 选出第一批最适合公开展示的内容
4. 用 Astro 做静态网站
5. 首页突出个人品牌和课程方向
6. AI工具导航作为重点栏目
7. 下载中心先展示目录，部分开放下载
8. 课程中心用于转化企业客户
9. 后续再加搜索、登录、会员、支付
10. 最后升级为AI知识库

18. 一句话总结

你的目标不是“做一个网站”，而是把多年积累的课程、工具、案例、模板和经验，整理成一个可以持续更新、可以搜索、可以转化客户、可以卖会员、未来还能接入 AI 助教的个人知识产品平台。
