## 命名规则

- 格式：`teacher-ai-weekly-report-{XX}.html`
- XX 为两位数字期号：01, 02, 03, ..., 10, 11, ...
- 所有周报统一存放在 `free/` 目录下

## 部署说明

- 此目录下的内容会随网站一起部署到 GitHub Pages
- 所有链接使用相对路径（如 `../index.html` 返回首页）
- 不包含任何需要付费或登录才能访问的内容

## 后续规划

- 会员版内容请放 `../member/` 目录（后续接入权限管理后启用）
